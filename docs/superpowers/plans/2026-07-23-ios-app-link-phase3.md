# iOS App ↔ Mac App Native Link — Phase 3 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let the iPhone ask the llm-ide agent a question over the paired mobile WebSocket, have the Mac app proxy it to the `:3456` backend (`/kb/agent/ask`), and stream the reply back to the iPhone's chat transcript. This establishes the typed command-channel pattern (SharedProtocol message → `MobileControlManager.handleInbound` dispatch → `LlmIdeAPIClient` → reply) that later commands reuse.

**Scope (narrowed from spec, flagged for approval):** Chat channel only. Deep-link / app-launch / menu-click control deferred to a tight **Phase 3b** (reuses this dispatch infrastructure). Image attachments ARE included (server already accepts `image`).

**Architecture:** iPhone sends a typed `LlmIdeChat` SharedProtocol message (flat: `{type, commandId, text, history, image?}`). `MobileControlManager.handleInbound` decodes it, calls `LlmIdeAPIClient.askAgent(message:history:image:)` (existing method, extended with `image`), and sends back an `Output{commandId, payload:{stream, done:true}}` whose nested-payload shape matches the iPhone's existing receive loop (so iOS receive code is unchanged). On failure it sends `CommandError{commandId, message}`. `LlmIdeAPIClient` is wired into `MobileControlManager` as a settable property (the app sets it at launch, same pattern as the existing `_projectStore` back-reference).

**Tech Stack:** Swift (macOS 14, iOS 16), SharedProtocol SPM package, Network.framework (existing server), XCTest.

**Continues from:** Phase 2 (commit `4d60600` on `main`). The native server (`MobileWebSocketServer`), `MobileControlManager.handleInbound` stub, and `askAgent` already exist.

## Global Constraints

(Verbatim values every task inherits.)
- New SharedProtocol `type` tags: `llmide_chat`, `output`, `error`. `Output` uses a nested `payload:{stream, done}` so it matches the iOS receive loop's existing `json["payload"]["stream"]` / `["done"]` reads — do NOT flatten it.
- `commandId` is a `String` (UUID generated by the iPhone); the Mac echoes it unchanged in `Output`/`CommandError`.
- `history` element shape: `{role:"user"|"assistant", content:String}`; `image` shape: `{mediaType:String, data:String}` (base64). These match `POST /kb/agent/ask`.
- `askAgent` keeps returning `String` (the server's single `{reply}`); no SSE/streaming added to the backend.
- Mac target: `@MainActor @Observable` for `MobileControlManager`; `swift-tools-version: 6.0`, `swiftLanguageModes: [.v5]` unchanged. `handleInbound` must run on MainActor (the `onInbound` callback hops via `Task { @MainActor in … }`).
- Do not change the iOS receive loop's dispatch on `type` strings (it already handles `output`/`ack`/`error`). Only the chat SEND path changes on iOS.
- Conventional Commits; one concern per commit; do not push.

```swift
func askAgent(message: String, history: [AgentAskMessage] = [],
              image: (mediaType: String, data: String)? = nil) async throws -> String {
    struct WireMsg: Encodable { let role: String; let content: String }
    struct WireImage: Encodable { let mediaType: String; let data: String }
    struct Req: Encodable {
        let message: String
        let history: [WireMsg]
        let image: WireImage?
    }
    struct Resp: Decodable { let reply: String }
    let wire = history.map { WireMsg(role: $0.role.rawValue, content: $0.content) }
    let img = image.map { WireImage(mediaType: $0.mediaType, data: $0.data) }
    let r: Resp = try await post("/kb/agent/ask",
                                 body: Req(message: message, history: wire, image: img),
                                 authenticated: true)
    return r.reply
}
```

```swift
/// Dispatch decoded inbound client messages. Phase 3 handles llm-ide chat;
/// deep-link/app/menu commands arrive in Phase 3b.
private func handleInbound(_ data: Data) {
    if let chat = try? JSONDecoder().decode(LlmIdeChat.self, from: data) {
        append(.info, "Chat: \(chat.text.prefix(40))")
        Task { await handleChat(chat) }
        return
    }
    let preview = String(data: data, encoding: .utf8) ?? "<\(data.count) bytes>"
    append(.info, "Unhandled inbound: \(preview)")
}

private func handleChat(_ chat: LlmIdeChat) async {
    guard let api else {
        await server?.send(CommandError(commandId: chat.commandId, message: "Backend not configured"))
        return
    }
    let history = chat.history.map { AgentAskMessage(role: .init(rawValue: $0.role) ?? .user, content: $0.content) }
    let image = chat.image.map { (mediaType: $0.mediaType, data: $0.data) }
    do {
        let reply = try await api.askAgent(message: chat.text, history: history, image: image)
        await server?.send(Output(commandId: chat.commandId,
                                  payload: OutputPayload(stream: reply, done: true)))
    } catch {
        append(.stderr, "askAgent failed: \(error.localizedDescription)")
        lastError = error.localizedDescription
        await server?.send(CommandError(commandId: chat.commandId, message: error.localizedDescription))
    }
}
```

```swift
func sendLlmideChat(_ text: String, image: (data: Data, mediaType: String)? = nil) {
    guard targetDevice != nil, connectionStatus == .connected else { return }
    let history = llmIdeMessages.suffix(10).compactMap { m -> ChatTurn? in
        guard !m.text.isEmpty else { return nil }
        return ChatTurn(role: m.role == .assistant ? "assistant" : "user", content: m.text)
    }
    llmIdeMessages.append(ChatMessage(role: .user, text: text, imageData: image?.data))
    llmIdeMessages.append(ChatMessage(role: .assistant, text: ""))
    llmStreaming = true
    let id = UUID().uuidString
    llmIdeCommandIds.insert(id)
    let img = image.map { ChatImage(mediaType: $0.mediaType, data: $0.data.base64EncodedString()) }
    let chat = LlmIdeChat(commandId: id, text: text, history: history, image: img)
    if let data = try? JSONEncoder().encode(chat),
       let str = String(data: data, encoding: .utf8) {
        sendTextFrame(str)
    } else {
        errorMessage = "Failed to encode chat message"
        disconnect(clearDirect: true)
    }
}
```
